# Spec 02: Frontend Scaffold

## Requirements
- Next.js 14 PWA with cyberpunk theme
- RainbowKit wallet connection (Base chain)
- All UI components render without errors
- wagmi hooks for contract interaction

## Acceptance Criteria
- [ ] `npm run build` succeeds without errors
- [ ] `npm run dev` starts dev server
- [ ] Lobby page renders with table card grid
- [ ] Wallet connect button works (RainbowKit modal opens)
- [ ] All UI components (Button, Card, Badge, Modal, GlowText) render
- [ ] TableCard, HeatIndicator, ProbabilityMonitor components exist
- [ ] Header with logo + wallet info renders
- [ ] wagmi hooks compile: useTableState, usePlaceBet, useClaimSeat, useAbandonSeat
- [ ] Cyberpunk theme colors applied (dark bg, neon accents)
- [ ] PWA manifest exists

**Output when complete:** `<promise>DONE</promise>`
